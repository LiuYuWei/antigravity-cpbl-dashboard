# CPBL 冬季聯盟儀表板

本專案是 CPBL (中華職棒大聯盟) 2025 冬季聯盟的儀表板。它以使用者友善的介面顯示戰績、賽程和其他資訊。

## 功能

-   **儀表板：** 聯盟概覽。
-   **戰績：** 顯示目前的球隊戰績。
-   **賽程：** 顯示比賽時程表。
-   **時間軸：** 賽事時間軸。

## 如何執行

此專案是使用 [Vite](https://vitejs.dev/) 建立的。

1.  **安裝依賴套件：**
    ```bash
    npm install
    ```
2.  **執行開發伺服器：**
    ```bash
    npm run dev
    ```
3.  在瀏覽器中開啟 `http://localhost:5173/` (或輸出中指定的埠號) 來查看應用程式。
